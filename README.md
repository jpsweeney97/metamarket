# metamarket

Meta-level tools for Claude Code development.

## Installation

Add the marketplace:

```bash
/plugin marketplace add jpsweeney97/metamarket
```

## Available Plugins

### skillosophy

Collaborative skill creation with deep methodology and multi-agent synthesis.

```bash
/plugin install skillosophy@metamarket
```

**Features:**
- Interactive dialogue for requirement exploration
- 4-agent synthesis panel for quality assurance
- Triage script for skill request analysis
- Reference materials and templates
